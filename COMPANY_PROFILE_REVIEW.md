# 기업 프로필 검토 결과

## 📋 현재 구현 상태

### ✅ 구현된 기능

1. **프로필 타입 구분**
   - `profile_type: 'COMPANY'`로 기업 프로필 구분
   - 프로필 전환 기능 (`ProfileSwitchButton`)
   - 사이드 네비게이터에서 기업 프로필별 메뉴 표시

2. **기업 프로필 전용 메뉴**
   - 내 프로젝트 목록 (`/enterprise/my-counsel`)
   - 관심 메이커 (`/my/bookmarked-makers`)

3. **프로필 표시**
   - 기본 프로필 정보 (username, bio, main_job, expertise)
   - 경력 인증 배지 표시
   - 연락 가능 여부 표시

4. **프로필 생성**
   - 기업 프로필 생성 기능 (`/my/profile/create`)
   - 기업 가입 시 자동 프로필 생성 (`EnterpriseAuthForm`)

---

## ⚠️ 문제점 및 개선 필요 사항

### 1. 기업 프로필에 불필요한 정보 표시

**문제점:**
- 기업 프로필에도 경력/학력/자격증 섹션이 표시됨
- 기업 프로필에는 개인 경력/학력/자격증이 필요 없음
- 기업 프로필에는 연락 가능 여부가 적절하지 않을 수 있음

**현재 코드:**
```jsx
// ProfileClient.jsx
<WorkExperienceMeta ... />  // 기업 프로필에도 표시됨
<EduCationMeta ... />        // 기업 프로필에도 표시됨
<LicenseMeta ... />          // 기업 프로필에도 표시됨
{profile_type === 'FREELANCER' && (
  <PortfolioMeta ... />      // 프리랜서만 표시 (정상)
)}
```

**권장 개선:**
```jsx
{profile_type === 'FREELANCER' && (
  <>
    <WorkExperienceMeta ... />
    <EduCationMeta ... />
    <LicenseMeta ... />
    <PortfolioMeta ... />
  </>
)}
{profile_type === 'COMPANY' && (
  <>
    {/* 기업 프로필 전용 섹션 */}
    <CompanyInfoMeta ... />  // 회사 정보
    <CompanyProjectsMeta ... />  // 진행 중인 프로젝트
  </>
)}
```

---

### 2. 기업 프로필 전용 정보 부족

**문제점:**
- 기업 프로필에 회사 정보가 부족함
- 사업자 등록번호, 회사 규모, 설립일 등 기업 정보 없음
- 기업 프로필에 적합한 필드가 없음

**필요한 정보:**
- 회사명 (현재 username으로 사용 중)
- 사업자 등록번호
- 회사 규모 (직원 수)
- 설립일
- 회사 주소
- 회사 홈페이지
- 주요 사업 분야
- 회사 소개 (현재 bio로 사용 중)

**권장 개선:**
- `accounts` 테이블에 기업 전용 필드 추가 또는 별도 테이블 생성
- 기업 프로필 편집 페이지에 기업 정보 입력 필드 추가

---

### 3. 연락 가능 여부 기능

**문제점:**
- 기업 프로필에도 "연락 가능/불가" 상태가 표시됨
- 기업 프로필에는 개인 연락 가능 여부보다는 영업 시간이나 응답 시간이 적절할 수 있음

**권장 개선:**
- 기업 프로필일 때는 연락 가능 여부 대신 "영업 중/영업 종료" 또는 "평균 응답 시간" 표시
- 또는 기업 프로필에서는 연락 가능 여부 기능 제거

---

### 4. 사이드 네비게이터 메뉴 부족

**현재 메뉴:**
- 내 프로젝트 목록
- 관심 메이커

**부족한 메뉴:**
- 프로젝트 상담 신청 (`/enterprise/counsel-form`) - 헤더에는 있지만 사이드 메뉴에 없음
- 팀 제안 현황 (`/my/team-proposal`) - 기업이 메이커에게 제안한 내역

**권장 개선:**
```jsx
// SideNavigator.jsx
if (isCompany) {
  return [
    {
      label: '내 프로젝트 목록',
      href: '/enterprise/my-counsel',
    },
    {
      label: '프로젝트 상담 신청',
      href: '/enterprise/counsel-form',
    },
    {
      label: '관심 메이커',
      href: '/my/bookmarked-makers',
    },
    {
      label: '팀 제안 현황',
      href: '/my/team-proposal',
    },
  ]
}
```

---

### 5. 프로필 편집 페이지

**문제점:**
- 기업 프로필 편집 페이지 (`/my/update`)가 프리랜서와 동일한 필드만 제공
- 기업 전용 필드가 없음

**권장 개선:**
- 프로필 타입에 따라 다른 편집 폼 표시
- 기업 프로필일 때는 기업 정보 입력 필드 표시

---

### 6. 프로필 표시 페이지

**문제점:**
- 기업 프로필을 볼 때도 프리랜서와 동일한 정보만 표시
- 기업 프로필에 적합한 정보가 부족

**권장 개선:**
- 기업 프로필 전용 표시 섹션 추가
- 진행 중인 프로젝트, 완료한 프로젝트 수 등 표시
- 회사 정보 섹션 추가

---

### 7. 북마크/제안 기능

**현재 상태:**
- 기업 프로필에서도 북마크/제안 버튼이 표시됨
- 기업 프로필은 메이커를 북마크/제안하는 주체이므로, 자신의 프로필에 이 버튼이 있는 것은 부적절

**권장 개선:**
- 기업 프로필일 때는 북마크/제안 버튼 숨김
- 또는 기업 프로필에는 다른 액션 버튼 표시 (예: 프로젝트 의뢰하기)

## 🔧 우선순위별 개선 사항

### 높은 우선순위 (즉시 개선 권장)

1. **기업 프로필에서 경력/학력/자격증 섹션 숨김**
   - 기업 프로필에는 개인 경력 정보가 불필요

   - 기업 프로필은 제안하는 주체이므로 자신의 프로필에 이 버튼 불필요
   - 코드 수정 필요: `ProfileClient.jsx`

5. **연락 가능 여부 기능 개선**
   - 기업 프로필일 때 다른 표시 방식 사용
   - 또는 기업 프로필에서는 제거

### 낮은 우선순위

6. **기업 프로필 전용 표시 섹션 추가**
