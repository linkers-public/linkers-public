# 프리랜서 사이드 네비게이터 검토 결과

## 📋 현재 구현 상태

### ✅ 현재 표시되는 메뉴

**프리랜서 프로필 (FREELANCER):**
1. 쪽지함 (`/my/messages`)
2. 할당된 프로젝트 (`/my/assigned-projects`)

**매니저 역할인 경우 추가:**
3. 팀 프로필 조회 (`/team-profile`)
4. 팀 프로젝트 확인 (`/team-projects`)

---

## ⚠️ 문제점 및 개선 필요 사항

### 1. 존재하지 않는 페이지 링크

**문제점:**
- `/team-profile` - 페이지 존재 여부 확인 필요
- `/team-projects` - 페이지 존재 여부 확인 필요

**권장 조치:**
- 페이지가 존재하지 않으면 메뉴에서 제거
- 또는 실제 페이지 생성

---

### 2. 누락된 중요한 메뉴

**헤더에는 있지만 사이드 메뉴에 없는 항목:**
- 프로젝트 찾기 (`/search-projects`) - 프리랜서가 프로젝트를 찾는 주요 기능
- 메이커 검색 (`/search-makers`) - 프리랜서가 다른 메이커를 검색하는 기능

**존재하는 페이지이지만 메뉴에 없는 항목:**
- 견적서 대시보드 (`/my/estimates-dashboard`) - 프리랜서가 제출한 견적서 관리
- 견적 요청 (`/my/estimate-requests`) - 받은 견적 요청 관리

**권장 개선:**
```jsx
// 프리랜서 프로필인 경우
if (isFreelancer) {
  const baseRoutes = [
    {
      label: '프로젝트 찾기',
      href: '/search-projects',
    },
    {
      label: '메이커 검색',
      href: '/search-makers',
    },
    {
      label: '쪽지함',
      href: '/my/messages',
    },
    {
      label: '할당된 프로젝트',
      href: '/my/assigned-projects',
    },
    {
      label: '견적서 대시보드',
      href: '/my/estimates-dashboard',
    },
    {
      label: '견적 요청',
      href: '/my/estimate-requests',
    },
  ]
  
  // 매니저 역할인 경우 팀 관련 메뉴 추가
  if (account?.role === 'MANAGER') {
    baseRoutes.push(
      {
        label: '팀 프로필 조회',
        href: '/team-profile',  // 존재 여부 확인 필요
      },
      {
        label: '팀 프로젝트 확인',
        href: '/team-projects',  // 존재 여부 확인 필요
      }
    )
  }
  
  return baseRoutes
}
```

### 3. 메뉴 구조 개선

**현재 구조:**
- 마이 섹션: 쪽지함, 할당된 프로젝트
- 팀 섹션: 팀 프로필 조회, 팀 프로젝트 확인 (매니저만)

**개선 제안:**
- 마이 섹션: 프로젝트 찾기, 메이커 검색, 쪽지함, 할당된 프로젝트, 견적서 대시보드, 견적 요청
- 팀 섹션: 팀 프로필 조회, 팀 프로젝트 확인 (매니저만)

또는 더 세분화:
- 탐색 섹션: 프로젝트 찾기, 메이커 검색
- 마이 섹션: 쪽지함, 할당된 프로젝트, 견적서 대시보드, 견적 요청
- 팀 섹션: 팀 프로필 조회, 팀 프로젝트 확인 (매니저만)

### 4. 메뉴 순서 최적화

**현재 순서:**
1. 쪽지함
2. 할당된 프로젝트

**권장 순서 (사용 빈도 고려):**
1. 프로젝트 찾기 (가장 자주 사용)
2. 할당된 프로젝트 (현재 진행 중인 프로젝트)
3. 견적서 대시보드 (제출한 견적서 관리)
4. 견적 요청 (받은 견적 요청)
5. 쪽지함 (메시지 확인)
6. 메이커 검색 (상대적으로 덜 사용)

   - 페이지는 존재하지만 메뉴에 없음

3. **견적 요청 메뉴 추가**
   - 받은 견적 요청을 관리하는 기능
   - 페이지는 존재하지만 메뉴에 없음

## 💡 추가 제안

1. **메뉴 아이콘 추가**
   - 현재 모든 메뉴에 아이콘이 비어있음 (`icon: ''`)
   - 각 메뉴에 적절한 아이콘 추가로 시각적 구분 향상

2. **활성 메뉴 표시 개선**
   - 현재는 `isActive`로 텍스트 색상만 변경
   - 배경색이나 아이콘 강조로 더 명확하게 표시

3. **메뉴 그룹화**
   - 관련 기능끼리 그룹화하여 가독성 향상
   - 예: 프로젝트 관련, 메시지 관련, 견적 관련
